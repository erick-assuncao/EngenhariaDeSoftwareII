Como Jogar Ludo Online

  • Inicialização do Jogo:
    • O jogo começa automaticamente quando a página é carregada.
    • Dois jogadores são representados como P1 e P2.
    • Cada jogador tem quatro peças localizadas em suas bases iniciais.
  • Jogar os Dados:
    • Para jogar os dados, clique no botão "Dado".
    • O valor dos dados será exibido na tela.
    • Se o valor for 6, você pode mover uma peça da base para a posição inicial.
  • Mover as Peças:
    • Após jogar os dados, clique em uma peça que está destacada para movê-la.
    • As peças só podem ser movidas se o valor dos dados permitir.
    • Se a peça está na base, você precisa de um 6 para movê-la para a posição inicial.
    • Se a peça está em movimento, ela se move o número de casas correspondentes ao valor dos dados.
  • Regras Especiais:
    • Se você jogar um 6, você tem direito a uma jogada extra.
    • As peças não podem se mover para a posição de outra peça da mesma cor.
    • Se uma peça terminar sua jogada na posição de uma peça adversária (fora das zonas seguras), a peça adversária retorna à base.
    • Cada jogador deve mover suas peças através do tabuleiro, completando uma volta e entrando na linha de chegada para alcançar a casa final.
  • Vencer o Jogo:
    • Para vencer, um jogador deve mover todas as suas quatro peças para a casa final (home position).
    • O primeiro jogador a alcançar a casa final com todas as suas peças vence o jogo.
  • Reset do Jogo:
    • Para resetar o jogo, clique no botão "Reset".
    • Todas as peças retornarão às bases iniciais, e o jogo recomeçará.
